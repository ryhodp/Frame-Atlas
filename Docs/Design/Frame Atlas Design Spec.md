# Frame Atlas Design Specification

## Overview
Frame Atlas is a premium visual reference library for cinematographers and directors. The aesthetic is "cinematic, quiet, and image-first," drawing inspiration from Arc Browser, Obsidian, and Apple Photos.

## Core Layout (Desktop)
- **Three-Column Architecture**:
  - **Left Sidebar**: Deep charcoal navigation (#171A20) with thin borders.
  - **Center Content**: Masonry grid with minimal spacing. High information density.
  - **Right Inspector**: Sliding metadata panel with detailed technical specs (Mood, Lighting, Film, DP, etc.).

## Key Design Tokens
- **Accent**: Warm Gold (#D9A441)
- **Backgrounds**: Deep Charcoal (#0F1115), Layered Panels (#171A20)
- **Borders**: Very low opacity, subtle structure.
- **Typography**: Manrope (Modern, architectural).

## Interaction Principles
- Subtle elevation and gentle scaling on hover.
- Glassmorphic overlays for tooltips and floating UI elements.
- Soft shadows and "Apple Pro App" level of polish.
